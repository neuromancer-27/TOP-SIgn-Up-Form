TOP Sign-Up Form

A responsive sign-up form built with HTML and CSS, created as part of The Odin Project curriculum to demonstrate core front-end development skills.

Overview

This project demonstrates the creation of a clean, modern form layout using semantic HTML and Flexbox-based CSS.
The focus is on structure, layout, and user interface design rather than JavaScript-driven functionality.

Key Features

- Semantic HTML structure

- Responsive Flexbox layout

- Custom-styled form inputs and buttons

- Clear visual hierarchy and spacing

- Desktop-first design with somewhat responsive adjustments

- Validation with HTML only no JavaScript
